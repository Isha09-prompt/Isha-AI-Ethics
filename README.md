# 🤖 AI Ethics RAG System using n8n

## 📌 Project Overview
This project demonstrates an **AI Ethics RAG (Retrieval-Augmented Generation) System** built using **n8n workflow automation**.  
It integrates document-based knowledge with AI to provide intelligent responses related to AI ethics.

---

## 🚀 Features
- 📄 Upload and process AI Ethics documents (PDF)
- 🔍 Retrieval-Augmented Generation (RAG) pipeline
- 🤖 Automated workflow using n8n
- 💬 Chat-based interaction system
- 📊 Visual workflow representation

---

## 🛠️ Tools & Technologies Used
- **n8n** – Workflow automation
- **AI Models** – For generating responses
- **RAG Architecture** – For contextual answers
- **GitHub** – Version control

## ⚙️ How to Use

1. Import the JSON file into n8n:
   - Open n8n
   - Click **Import Workflow**
   - Upload `AI Ethics RAG System Chatflow (1).json.txt`

2. Add your API keys (if required)

3. Run the workflow

4. Interact with the chat system
